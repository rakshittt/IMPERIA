# IMPERIA Software Analysis Agents

IMPERIA uses software analysis modules. They are not humans and should not pretend to be humans.

Stock-first research can use these roles:

| Agent/module | Responsibility |
| --- | --- |
| News Analyst | Recent company news, events, article sentiment, citations |
| Fundamentals Analyst | Valuation, growth, margins, balance sheet, cash flow |
| SEC Filings Analyst | 10-K, 10-Q, 8-K, DEF 14A, Form 4, 13F-related context |
| Earnings Analyst | Next earnings, surprise history, beat/miss rates |
| Market Sentiment Agent | Price, volume, news, earnings, sector, and optional prediction-market signals |
| Macro/Sector Agent | Broad market and sector context |
| Bull Thesis Agent | Strongest reasonable bullish research case |
| Bear Thesis Agent | Strongest reasonable bearish research case |
| Risk Analyst | Business, financial, valuation, regulatory, execution, and market risks |
| Research Synthesizer | Final cited report or concise answer |

Dataflows gather evidence first. Agents analyze the evidence and return structured outputs with citations and warnings.
