# IMPERIA Demo Universe

The demo universe is stored at:

```text
tradingagents/data/demo/demo_universe.json
```

It is a practical educational set of popular large-cap US-listed companies and major ETFs for capstone demos. It is not claimed to be a live official top-100 ranking.

Demo mode supports quote, profile, metrics, news, earnings, filing, sentiment, and research-report fixtures for each ticker through deterministic fixture generation.
